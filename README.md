### Portfólio

<div align="center">

**Projeto — Classificador Fruta vs Vegetal**  
**Stack:** Python · Flask · PostgreSQL (+ pgvector) · PGVector · Docker  
**Descrição:** Serviço simples que usa embeddings e distância por cosseno para classificar itens como *fruta* ou *vegetal*. API REST para enviar texto/imagem (descrição) e receber a predição com score de similaridade.  
**Destaque:** protótipo orientado a produção — indexação vetorial em Postgres, endpoint de inferência e testes automatizados.  
**Link:** https://github.com/seu-usuario/projeto-fruta-vegetal
